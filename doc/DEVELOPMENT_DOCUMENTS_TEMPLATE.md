## 📋 文档模板

### 文档结构模板
```markdown
# 文档标题

## 概述
简要说明文档目的和范围

## 前置条件
阅读本文档需要的基础知识

## 详细内容
具体的文档内容

## 示例
相关示例和代码

## 注意事项
重要的提醒和警告

## 参考资料
相关文档和链接
```

### API文档模板
```markdown
# API文档

## 概述
API功能说明

## 认证方式
认证机制说明

## 端点列表
### [METHOD] /path
**描述**: 接口功能说明
**参数**:
- `param1` (类型, 必填) - 参数说明
- `param2` (类型, 可选) - 参数说明

**响应**:
```json
{
  "status": "success",
  "data": {}
}
```

**错误码**:
- 400: 参数错误
- 401: 未授权
- 500: 服务器错误
```

## 🔄 文档维护流程

### 创建新文档
1. 确定文档类型和目录
2. 使用对应模板创建文档
3. 填写文档内容
4. 更新README.md索引
5. 提交代码审查

### 更新现有文档
1. 确定需要更新的部分
2. 修改文档内容
3. 更新版本号和日期
4. 记录变更内容
