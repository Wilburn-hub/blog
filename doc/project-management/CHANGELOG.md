### 技术栈
- **前端**: Next.js 14, TypeScript 5.6, Tailwind CSS 3.4
- **后端**: Next.js API Routes, Prisma 5.21
- **数据库**: PostgreSQL 15, Redis 7
- **认证**: NextAuth.js 4.24
- **部署**: Docker, Docker Compose, Nginx
- **CI/CD**: GitHub Actions
- **监控**: 自定义监控脚本
- **测试**: Jest, Playwright

## 版本说明

### 版本号规则
- **主版本号**: 不兼容的API修改
- **次版本号**: 向下兼容的功能性新增
- **修订号**: 向下兼容的问题修正

### 更新类型
- `新增` - 新功能
- `更改` - 对现有功能的变更
- `弃用` - 即将移除的功能
- `移除` - 已移除的功能
- `修复` - 问题修复
- `安全` - 安全相关的修复

### 发布周期
- **主版本**: 根据需要发布
- **次版本**: 每月发布
- **修订版**: 根据需要发布
